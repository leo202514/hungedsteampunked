# Hunged: A Steampunk Hangman Trial

A Victorian industrial themed hangman game built with React and Bootstrap.

## Rules of Engagement

1. The computer selects a random mechanical term (e.g., AUTOMATON).
2. The user clicks letters on the typewriter interface.
3. Correct guesses reveal letters in the word display.
4. Incorrect guesses assemble the gallows automaton.
5. The game ends in victory if the word is completed, or defeat if 6 mistakes are made.

## Installation Instructions

1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Clone or download this project folder.
3. Open your terminal/command prompt in the project root.
4. Run `npm install` to download dependencies.
5. Run `npm start` to launch the game in your browser.

## Technical Features

- Built with **React** and **React-Bootstrap**.
- Responsive layout using **Flexbox** and Grid.
- Custom Steampunk assets and Google Fonts.
